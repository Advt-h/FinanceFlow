# Technologies and Frameworks Used in This Project

## 1. React (Advanced)
- UI library for building component-based user interfaces.
- Used with hooks (useState, useEffect), functional components, and JSX/TSX.
- Routing handled with react-router-dom.

## 2. TypeScript (Intermediate/Advanced)
- Superset of JavaScript adding static typing.
- Used throughout the project for type safety and better developer tooling.

## 3. Vite (Intermediate)
- Fast build tool and development server for modern web projects.
- Used for local development and production builds.

## 4. Firebase (Intermediate)
- Authentication: Email/password sign-up, login, and logout.
- Firestore: NoSQL database for storing user profiles and expenses.
- Modular SDK (v9+), not legacy namespaced API.

## 5. Recharts (Basic)
- Charting library for React.
- Used for rendering the expenses pie chart.

## 6. Tailwind CSS (Basic)
- Utility-first CSS framework.
- Used via CDN for rapid styling with utility classes.

## 7. Lucide React (Basic)
- Icon library for React.
- Used for UI icons (sidebar, buttons, etc).

## 8. Node.js & npm (Basic)
- Node.js: JavaScript runtime for running build tools and dev server.
- npm: Package manager for installing dependencies.

---

### Usage Levels
- **Advanced:** React, TypeScript (custom types, hooks, state management, component composition)
- **Intermediate:** Vite, Firebase (auth, Firestore, modular SDK)
- **Basic:** Recharts, Tailwind CSS (via CDN), Lucide React, Node.js/npm
